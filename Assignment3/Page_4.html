Don'tDrive's static webpage.

This is the code repository for Don'tDrive's promotional website. Don'tDrive is a ride-sharing startup founded in Reno, Nevada. The site is compiled to static javascript via GatsbyJS. Production builds are continuously deployed to https://www.dontdrive.life via Netlify(the splash branch) . Development builds are deployed to https://master--dontdrive.netlify.com/(currently the master branch)

Previous Iterations

This repository is essentially a react port of chauffeur-ride-request, a jQuery implementation of the ride request screen, chauffeur, a static html/css implementation of the landing page, and chauffeur-wp, a wordpress implementation of the landing page, as well as an attempt to adhere to a cleaner development workflow.

Built With

  • GatsbyJS - Content Management System, React based
  • gatsby-starter-strict - Linting & autoformatting boilerplate for Gatsby
  • Netlify - Continuous deployment
  • react-player - React video component
  • Google Maps API - Ride request geolocation (may switch to alternate provider for at launch)
  • jQuery - DOM Manipulation
  • jQuery Mask Plugin - Masking ride request form field
  • Parsley - Ride request Form validation
